The BC Government and the broader public service sector in the province collectively operate many RESTful web services (also known as Application Programming Interfaces, or APIs). This web site provides a means to register an API with the BC Data Catalog and to optionally opt in to DataBC's API gateway, which can provide additional technical capabilities on top of an existing API.

OpenAPI specification

OpenAPI is a standard format for specifying the capabilities of a RESTful API.

Although you may register an API without an OpenAPI specification, it is good practice for all RESTful APIs to provide a specification file in a standard format, such as the OpenAPI format.

API gateway

DataBC offers APIs in the BC public service access to an API gateway. The API gateway can add additional capabilities (such as access control and throttling) on top of an API's core business functions. In addition, APIs registered with the API gateway can be assigned a conveinent URL (https://app-short-name.api.gov.bc.ca).

Use of DataBC's API gateway is optional. If the API being registered here already uses a different API gateway, or if you otherwise do not require the capabilities of an API gateway, you may opt out.
